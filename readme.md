# Django Bookstore API

A simple Django REST-style API for managing bookstore lending records.  
The API allows creating, retrieving, updating, and deleting book records using Django ORM.

The project focuses on learning:

- Django API development (without templates)
- CRUD operations
- Django ORM
- QuerySets
- Decorators
- Testing APIs with Postman

---

## Features

- Create a book record
- Retrieve all books
- Retrieve a specific book
- Update book details
- Delete a book

Each record stores:

- Book ID
- Book Name
- Lender Name
- Phone Number
- Date Given
- Date Returned
- Return Status

## Tech Stack

- Python
- Django
